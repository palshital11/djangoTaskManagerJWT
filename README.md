# djangoTaskManagerJWT

A secure and scalable **Task Manager REST API** built using Django REST Framework and JWT authentication (SimpleJWT).  
Users can register, log in, and manage their personal tasks with full CRUD operations.  
Designed as a clean and modular backend suitable for learning, internships, and production-level fundamentals.
